# Laporan Programming Assignment 1: Basic C++

## Deskripsi Program
Program ini diperuntukkan untuk menghitung umur seseorang dalam tahun dan bulan serta di hari ia lahir.

## Input dan Output
Input yang diketik dalam terminal atau konsol adalah tanggal dalam format ```DD/MM/YY```. Output yang akan keluar adalah ```[Tahun] [Bulan] [Hari Lahir]```. 

> Contoh 1
```
20/02/2006 //masukkan
20 241 Senin //keluaran
```

>> Contoh 2
```
21/10/1993 //masukan
32 390 Kamis //keluaran
```
